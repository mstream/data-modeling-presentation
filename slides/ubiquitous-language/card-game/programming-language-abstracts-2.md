# Domain code should **not** use programming language abstracts

<br/>

| bad | | good |
| ---: | :---: | :--- |
| Float `2.499` | vs | Price `2.49` |
| Int `-13` | vs | Age `13` |
| String `john` | vs | Name `John` |
| Char `%` | vs | Initial `J` |
| **Validation** | vs | **Parsing** |

<br/>
